# Tutoring E-Commerce Platform

A fully responsive e-commerce website for connecting students with professional tutors. 
Built with HTML, CSS, JavaScript, and Bootstrap 5.

## 🎓 Features

- **Homepage**: Hero section with featured courses and tutors
- **Product Catalog**: Browse educational services across multiple categories
- **Shopping Cart**: Add/remove items with real-time updates
- **User Authentication**: Secure login and sign-up system
- **Payment Integration**: Support for Visa, Mastercard, EasyPaisa, and JazzCash
- **Tutor Profiles**: Detailed information about available instructors
- **Contact System**: Customer support and inquiry forms
- **About Us**: Company mission and values
- **Responsive Design**: Mobile-friendly interface for all devices

## 🚀 Getting Started

1. Clone this repository
2. Open `Main Page.html` in your web browser
3. Navigate through pages using the navigation menu
4. Explore products, add to cart, and proceed to checkout

## 🛠️ Technologies Used

- **HTML5**: Semantic markup
- **CSS3**: Custom styling and responsive layouts
- **Bootstrap 5**: Responsive framework
- **JavaScript**: Interactive features
